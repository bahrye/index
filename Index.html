Selamat Datang di Pusat Bantuan!

Panduan ini dirancang untuk membantu Anda menguasai setiap fitur aplikasi dengan cepat dan mudah. Mari kita mulai perjalanan Anda dalam mengolah nilai ijazah secara digital!

Panduan Cepat: Dari Awal Hingga Cetak Ijazah

1

Lengkapi Data Sekolah

Langkah paling pertama dan utama. Buka menu Data Sekolah.

Pastikan semua informasi terisi dengan benar, terutama Jenjang Sekolah (MI/MTS/MA), karena ini akan menentukan menu-menu selanjutnya.

Isi semua data sekolah lalu simpan.

2

Atur Mata Pelajaran & Bobot Nilai

Buka menu Mapel & Bobot.

  • Gunakan tombol "Import Mapel by System" untuk memuat daftar mapel standar secara otomatis.
  • Atur persentase "Bobot Nilai Ujian" dan "Bobot Nilai Rapor". Pastikan totalnya 100%.

Import mapel by system.

3

Isi Data Siswa

Buka menu Data Siswa. Anda bisa menambah siswa satu per satu atau untuk efisiensi, gunakan fitur "Upload Data Siswa" menggunakan template Excel yang disediakan.

Mengisi daftar nama siswa.

4

Isi Semua Nilai

Ini adalah bagian inti. Lanjutkan ke menu Input Nilai Ujian dan Input Nilai Rapor untuk semua semester. Anda juga bisa menggunakan fitur Upload Nilai untuk impor massal dari Excel agar lebih cepat.

Upload file nilai excel.

5

Proses & Lihat Hasil Akhir

Langkah terakhir! Buka menu Rekapitulasi, lalu klik tombol "Perbarui & Muat Ulang Data".

Aplikasi akan memproses semua nilai yang telah Anda input dan menampilkan hasil akhirnya. Dari sini Anda bisa mengekspor laporan dalam bentuk PDF atau Excel.

Pertanyaan yang Sering Diajukan (FAQ)

T: Mengapa pada tabel rekapitulasi, nilai siswa saya kosong atau tidak update?

J: Pastikan Anda telah menekan tombol berwarna hijau Perbarui & Muat Ulang Data di halaman Rekapitulasi. Tombol ini berfungsi sebagai "tombol proses" yang akan mengkalkulasi ulang semua nilai dan menampilkannya di tabel rekap.

T: Apa perbedaan antara fitur "Backup & Restore" dengan "Arsip Tahunan"?

J: Keduanya penting, namun tujuannya berbeda.

  • Backup & Restore: Gunakan ini di tengah-tengah pengerjaan untuk membuat "save point" atau cadangan data. Jika terjadi kesalahan, Anda bisa kembali ke data di titik backup tersebut.
  • Arsip Tahunan: Gunakan ini HANYA di akhir tahun ajaran. Fitur ini akan mengunci semua data tahun tersebut secara permanen (hanya lihat) dan membersihkan aplikasi agar siap digunakan untuk tahun ajaran baru.

T: Bagaimana cara melihat kembali data dari tahun ajaran yang sudah saya arsipkan?

J: Anda bisa mengaksesnya melalui menu Arsip Tahunan. Di sana Anda bisa melihat daftar semua arsip, membukanya dalam mode "hanya lihat", dan mengekspor datanya jika diperlukan.

T: Apakah data saya aman?

J: Ya, setiap akun sekolah memiliki file database-nya sendiri yang terpisah dan tidak dapat diakses oleh sekolah lain. Untuk keamanan ekstra, sangat disarankan untuk menggunakan fitur Backup & Restore secara berkala.
